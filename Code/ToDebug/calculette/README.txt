Calculator with Graphing Functionality

This is a Python-based calculator with a graphical user interface built using Tkinter. It also includes a feature to plot mathematical functions.

Setup
-----

1. Make sure you have Python 3 installed on your system.

2. Install the required libraries using pip:
   pip install -r requirements.txt

Running the Application
-----------------------

To run the calculator, execute the following command in your terminal:

python calculator.py
